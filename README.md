# Car Management System - Minimal API

A simple **Car Management System** built using **ASP.NET Core Minimal API** and **Entity Framework Core**.  
This project demonstrates **CRUD operations** with proper **DTOs**, **IResult responses**, and follows **RESTful API conventions**.

---

## **Features**

- **Get all cars** – Retrieve all cars in the database
- **Get car by ID** – Retrieve a specific car by its ID
- **Add a car** – Add a new car to the database
- **Update a car** – Update car details by ID
- **Delete a car** – Delete a car by ID
- Uses **DTOs** to structure API responses
- Proper **HTTP status codes** (`200 OK`, `201 Created`, `404 Not Found`, `400 Bad Request`)

---

## **Tech Stack**

- **Backend:** ASP.NET Core 8.0 Minimal API
- **Database:** Entity Framework Core (InMemory )
- **Language:** C#
- **Tools:** VS Code, Postman / Insomnia (for testing API endpoints)
